# design-systems-playground

A minimal playground configured for:

- Next.js 15 (App Router)
- TypeScript (strict mode)
- Tailwind CSS v4
- pnpm
- Framer Motion
- Radix UI / shadcn-style headless primitives
- Lucide React

## Development

```bash
pnpm install
pnpm dev
```

## Validation

```bash
pnpm lint
pnpm build
```
